# Andrew Isaak website package

This is a Quarto website package prepared for GitHub Pages.

## Quick start

1. Upload all files in this folder to your GitHub repository `andrewisaak.github.io`
2. Install Quarto on your computer
3. In the website folder, run:

   quarto render

4. Commit and push the generated `docs/` folder
5. In GitHub:
   - Settings
   - Pages
   - Source: Deploy from branch
   - Branch: main
   - Folder: /docs

## Notes

- The visuals in `images/` are clean placeholders so the site has a consistent visual structure immediately.
- If you want to replace them with official institutional logos later, use the same filenames.
- Add your CV PDF into the `files/` folder and update links if needed.
